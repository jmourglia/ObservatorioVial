4. Siniestros por hora del dia

df$siniestro_hora <- hms::as_hms(df$siniestro_hora)
df <- df %>%
  mutate(hora = hour(siniestro_hora))
df %>%
  count(hora) %>%
  ggplot(aes(x = factor(hora), y = n)) +
  geom_bar(stat = "identity", fill = "steelblue") +
  geom_text(aes(label = n), vjust = -0.5, color = "black") +
  labs(title = "Cantidad de siniestros por hora del da", 
       x = "Hora del da", 
       y = "Cantidad de siniestros") +
  theme_classic() +
  scale_x_discrete(breaks = 0:23) +
  theme(plot.title = element_text(hjust = 0.5, size = 16, face = "bold"))

A lo largo del anlisis de los siniestros viales en la ciudad de Corrientes durante 2023, se identificaron varios patrones:

Siniestros por zona de ocurrencia:* El grfico de barras revela que algunas zonas de la ciudad registran un mayor nmero de siniestros. Este resultado es til para identificar reas crticas donde se deben concentrar esfuerzos de seguridad vial, incrementando la presencia de controles de trnsito.
